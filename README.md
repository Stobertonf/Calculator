# Calculator

Projeto do curso da carreira de Backend .Net / Fundamentos do C#, do Professor André Baltieri (Balta), na plataforma https://balta.io para aprender a fazer uma calculadora. 

Neste curso eu tive a oportunidade de revisar os conceitos / Fundamentos de C#.

Além disso, tive a oportunidade de trabalhar com os operadores aritméticos (Soma, Subtração, Multiplicação e Divisão) de rever conceitos como Console.WriteLine,
Console.ReadLine e Console.ReadKey(). E por fim, tive a oportunidade de aplicar o switch case. 

O projeto foi finalizado por volta de 13/02/2022.
